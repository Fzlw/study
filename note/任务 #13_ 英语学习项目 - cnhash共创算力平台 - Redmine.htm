英语学习项目:

学多少，在自己

项目队伍不为个人的学习效果负责，学多少，在自己；
项目队伍队伍的唯一目的，是帮助各位更好地坚持；

任务目标： 每天学习半个小时英语，半年后（过年前）能看懂计算机，开发技术文档，区块链相关文档

项目成员：
PM: 陈香艳
成员： 李伟，陈香艳，徐子梁。。。。
抓手：

  1. 每天花半小时 学习英语，并将重点单词，句子，解释在本贴每天回复学习的内容：
    格式：

    1. repository 仓库; 贮藏室; 博物馆; 亲信;
      • Do you have a GitHub repository? 你有在github上创建仓库么
      • （更多例名）
    2. blockchain 区块链
      • Blockchain technology is cool ！区块链技术太厉害了
      • What is Blockchain Technology！ 区块链技术是啥玩样？

  2. 每周5周会大家过一遍

学习参考资料

  1. 网上自己搜
  2. https://eggjs.org/en/index.html
